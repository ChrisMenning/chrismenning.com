A cyberdeck built into a genuine cigar box. Raspberry Pi inside, orange mechanical keyboard bolted to the lid, dual small screens, hazard tape on the hinges because of course there is.

Runs a full Linux desktop. The aesthetic is part of the point — it's a functional computer that looks like it was built by hand, because it was.
